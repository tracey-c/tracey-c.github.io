1.root用户了解

root： 默认管理员(超级管理员) 最大权限
learn04： 机器名称 pwd 查看当前所在路径
~：当前该用户的家目录用 /root
# 代表的当前用户为root
$：代表当前是普通用户操作

2.ls查看命令

ls 显示文件夹及文件
ls -l 长格式 显示权限 文件数 用户 用户组 文件大小 时间 文件名
ls -l -a 显示隐藏文件 文件夹
ls -l -h 仅仅查看文件大小
ls -l -r -t 按时间排序（快速找到排序文件-r sort -t time）
等价
ls -1 <=> ll
ls -l -a <=> ll -a
ls -l -h <=> ll -h
ls -l -r -t <=> ll -rt

3.mkdir 创建目录命令

mkdir [-p] dirName
mkdir dir1 dir2 dir3 创建三个目录（并联创建）
mkdir -p dir4/dir5/dir6( 串 级联创建) ,加-p原本目录下创建目录（如无原本目录，同时创建原本目录）

4.cd切换目录路径命令

cd / 切换到根目录
cd …/ 回到上一级目录
cd …/…/ 返回上两级目录
cd - 回退到上一次的目录
cd ~切换到用户主目录
root用户家目录切换三种方法：
* cd /root
* cd 直接回车 (推荐）
* cd ~
普通用户家目录切换三种方法：
* cd /home/xx
* cd 直接回车
* cd ~

5.help命令帮助

[commmand] --help 命令帮助查询使用参数
eg：ls --help Usage: ls [OPTION]… [FILE]… []标识的 可选 … 多个参数
